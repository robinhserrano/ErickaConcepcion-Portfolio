## 📐 Interface Specifications
The portfolio utilizes a sophisticated design system defined in the `tailwind.config`:

*   **Primary Palette:** `#14b8b8` (Cyan Neon) / `#d946ef` (Secondary Magenta).
*   **Surface Logic:** High-blur glassmorphism (20px) on `surface-dark` backgrounds.
*   **Grid System:** Fixed 60px adaptive grid with pulsing animation.
*   **Overlay Logic:** 15% opacity CRT scanline layer for retro-futurist depth.
